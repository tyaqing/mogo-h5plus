# MogoH5+ 多页面网页解决脚手架

结合Hbuilder使用,可以快速开发 WebApp 

- `支持 Npm 生态`
- `支持 vue 语法,以及 vue 生态,比如 vux,mint,vant`
- `支持 ES6/ES7 语法`
- `使用 VConsole 调试`
- `VSCode 友好`
- `局域网便捷调试,不插数据线也可以调试`
- `兼容部分 mui 语法`

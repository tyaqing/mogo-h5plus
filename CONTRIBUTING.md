# Contributing

Thanks for helping maintain MogoH5+.

This repository is being revived as a legacy Vue 2 and HBuilder/H5+ scaffold. Contributions that improve documentation, dependency hygiene, compatibility notes, security review, and migration guidance are especially useful.

## Before Opening a Pull Request

Please include:

- What problem the change solves
- Whether it affects existing HBuilder/H5+ projects
- Node.js and npm versions used for verification
- Any known compatibility risk for Vue 2, Webpack 3, MUI, Vux, Vant, or HBuilder

## Good First Contributions

- Fix broken documentation links
- Add notes for modern Node.js setup issues
- Document known dependency vulnerabilities and workarounds
- Add reproduction cases for build or runtime issues
- Improve migration notes from old project layouts

## Compatibility

Avoid large dependency upgrades in the same pull request as behavior changes. For legacy users, small and reviewable changes are easier to validate.
